# Car Rental Project

## Getting Started

### Install packages

```sh
python -m venv venv
.\venv\Scripts\activate
pip3 install -r requirements.txt
```

### Run app.py

```sh
# Execute virtualenv (Windows)
.\venv\Scripts\activate

# Run app.py
python app.py
```
